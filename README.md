用于构建企业定制版的rpm包

# rpm 定制规范

* 安装文件统一到指定目录   例如： /lang/redis /lang/dataredis /lang/datapg  /lang/datamysql  /lang/
* 使用企业定制的前缀路径   例如： /test
* 支持主流的中间件、数据库  
* 自动化拉取最新的源代码
* 自动化生成rpm包
* 自动测试部署rpm包
* 支持主流操作系统

# 待支持组件
